## 1. ツールの概要

Instagramのキャンペーン投稿に対して、**いいね・コメントしたユーザーを自動抽出**し、フォロワー数でフィルタリングした候補者リストをCSVで出力するツールです。

**できること：**

- 指定した投稿の「いいね」したユーザー一覧の取得
- 指定した投稿の「コメント」したユーザー一覧の取得
- 各ユーザーのフォロワー数の自動取得
- 最低フォロワー数によるフィルタリング（例：1,000人未満を除外）
- 結果のCSVダウンロード

## 3. 初回セットアップ：アカウント登録

ツールを使うには、まずInstagramアカウントのセッションIDを登録する必要があります。

### 3-1. セッションIDの取得手順（Chrome）

1. Chromeで **instagram.com** を開き、キャンペーン用アカウントでログイン
2. **F12**キー（Macは `Cmd + Option + I`）で開発者ツールを開く
3. 上部メニューの **「Application」** タブをクリック
4. 左メニューの **「Cookies」→「https://www.instagram.com」** をクリック
5. 一覧から **「sessionid」** を探し、**Value列の値をコピー**

### 3-2. アカウントの登録

1. 画面上部の **「⚙️ アカウント管理」** タブを開く
2. 下部の **「➕ 新規アカウント追加」** セクションに以下を入力：

| 項目 | 入力内容 | 例 |
|------|---------|-----|
| 表示名（社内用） | チーム内で識別しやすい名前 | クライアントA |
| Instagramユーザー名 | @なしのユーザー名 | ig_account_name |
| セッションID | 上記手順でコピーした値 | （長い英数字の文字列） |

3. **「追加する」** ボタンをクリック

## 4. キャンペーン当選者を選定する手順

### ステップ1：抽出条件を設定する

1. **「📊 抽出ツール」** タブを開く
2. 以下の項目を設定：

| 項目 | 説明 |
|------|------|
| 使用するアカウント | 登録済みのアカウントから選択 |
| 投稿URL | キャンペーン対象の投稿URLを貼り付け（`https://www.instagram.com/p/XXXXXXXXX/` の形式） |
| 抽出モード | 下記の3パターンから選択 |
| 最低フォロワー数 | この数値未満のユーザーを除外（デフォルト：1,000） |

#### 抽出モードの選び方

| モード | 用途 |
|--------|------|
| **いいね＋コメント（どちらかでOK）** | いいねまたはコメントのどちらかをしたユーザーを全員抽出。最も広い候補リストが得られる |
| **いいねのみ** | いいねしたユーザーだけを抽出。コメント不要のキャンペーン向け |
| **コメントのみ** | コメントしたユーザーだけを抽出。コメント内容もCSVに出力される |

### ステップ2：抽出を実行する

1. 設定を確認して **「抽出開始」** ボタンをクリック
2. 「ジョブを登録しました」と表示されたら成功
3. **「📋 ジョブ一覧」タブに移動**して進捗を確認

> 処理はバックグラウンドで実行されます。ブラウザを閉じずにお待ちください。
> 対象ユーザーが多い場合、フォロワー数取得に時間がかかります（数百人で10〜30分程度）。

## 5. ジョブの確認・CSV ダウンロード

1. **「📋 ジョブ一覧」** タブを開く
2. ジョブのステータスを確認：

| ステータス | 意味 |
|-----------|------|
| 🕐 待機中 | 処理待ち |
| ⏳ 実行中 | 処理中（ログが自動更新されます） |
| ✅ 完了 | 処理完了。CSVダウンロード可能 |
| ❌ エラー | 処理失敗。エラー内容を確認 |

3. ✅ 完了になったら、**「📥 CSV」** ボタンをクリックしてCSVをダウンロード
4. 不要なジョブは **「削除」** ボタンで削除可能

## 6. CSV の見方と当選者選定の進め方

### CSVの列

| 列名 | 内容 |
|------|------|
| id_name | ユーザー名（@付き） |
| profile_url | Instagramプロフィールページへのリンク |
| follower_count | フォロワー数 |
| comment | コメント内容（コメント抽出モード時のみ） |

> CSVはフォロワー数の多い順にソートされています。

### 当選者選定の推奨フロー

1. **CSVをダウンロード**してスプレッドシートで開く
2. **応募条件の確認**：フォロー・いいね・コメントなどキャンペーン参加条件を満たしているか確認
3. **候補者の絞り込み**：フォロワー数・コメント内容などをもとに候補を選定
4. **プロフィール確認**：`profile_url` からプロフィールを開き、アカウントの質（投稿内容・エンゲージメント等）を目視確認
5. **当選者リスト確定**：チーム内で最終確認し、当選者を決定
6. **当選連絡**：DMなどで当選者へ連絡

## 7. アカウント管理（セッションID更新・削除）

### セッションIDの更新

セッションIDには有効期限があります（数ヶ月）。「セッション切れ」エラーが出たら更新が必要です。

1. **「⚙️ アカウント管理」** タブを開く
2. **「🔑 セッションID更新」** セクションで対象アカウントを選択
3. 新しいセッションIDを入力して **「更新する」** をクリック

### アカウントの削除

1. アカウント一覧から対象アカウントの **「削除」** ボタンをクリック

---

## 8. よくあるエラーと対処法

| エラー | 原因 | 対処法 |
|--------|------|--------|
| **セッション切れ** | セッションIDの有効期限切れ | 「アカウント管理」タブからセッションIDを再取得・更新 |
| **投稿URLエラー** | URLに `/p/` が含まれていない | 正しい投稿URLを確認して再入力 |
| **アカウント未登録** | アカウントが登録されていない | 「アカウント管理」タブからアカウントを追加 |
| **フォロワー数が空欄** | API制限でフォロワー数を取得できなかった | 時間を空けて再実行 |

## 注意事項

- **セッションIDは機密情報です。** Slack等で共有せず、ツール上で直接入力してください。
- **Instagram APIの制限**により、短時間に大量のリクエストを送るとアカウントが一時制限される可能性があります。連続実行は避けてください。
- **複数人が同時に別のジョブを実行できます。** ジョブはキュー管理されています。
